# AI Debugging Assistant

AI Debugging Assistant is a web-based AI agent that helps programmers
identify Python errors and understand how to fix them.

## Features

Detects Python errors
Explains why the error occurs
Suggests possible fixes
Provides debugging tips

## Technologies Used

Python
FastAPI
HTML
CSS

## How to Run

Install requirements

pip install -r requirements.txt

Run server

uvicorn app:app --reload

Open browser

http://127.0.0.1:8000
